# CampusPilot

Offline-first student productivity super app (PWA) with timetable, assignments, attendance, notes, Pomodoro, analytics, Firebase sync, and Groq AI planning.

## Quick Links

- Main project folder: [Campuspilot](./Campuspilot)
- Live app: [campuspilot-weld.vercel.app](https://campuspilot-weld.vercel.app)
- GitHub Pages: [lekhanpro.github.io/CampusPilot](https://lekhanpro.github.io/CampusPilot/)
- Full documentation: [Campuspilot/README.md](./Campuspilot/README.md)

## Start Locally

```bash
cd Campuspilot
npm install
npm run dev
```
