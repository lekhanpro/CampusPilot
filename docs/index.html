CampusPilot

Offline-first student productivity super app.

Offline-first student productivity super app (PWA) with timetable, assignments, attendance, notes, Pomodoro, analytics, Firebase sync, and Groq AI planning.

Open Live App View Repository

Core Modules

  • Timetable and calendar
  • Assignments and notes
  • Attendance tracker
  • Pomodoro and exam countdowns
  • Analytics and progress insights
  • Groq-powered AI planning

Architecture

  • Next.js 15 + TypeScript + Tailwind
  • Dexie/IndexedDB for local-first storage
  • Firebase Auth + Firestore for cloud sync
  • Sync queue with LWW conflict strategy
  • PWA with installable offline shell
